    The process:

    1. Edit the template.yaml:
    1. sam build to create the build version of the lambda function
    1. Sam deploy , which will create a stack in cloud-formation if not created before which will be used to manage the lambda function.
    1. Repeat step 2, 3 for redeploying or updating the lambda function.
